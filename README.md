# Eon

Brand and shop site for Eon, a New Zealand candle label pairing hand-poured soy candles with
crystals and a calm, wellness-led identity.

**Live:** https://eon-ht82.vercel.app

## Stack

- Next.js / React
- Multi-page build: home, shop, and about views
- Responsive product layout, deployed on Vercel

## Local development

```bash
npm install
npm run dev
```
